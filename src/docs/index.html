Troubleshooting

If you're experiencing issues with the Swagger UI (blank screen), try the following:

  1. Make sure NODE_ENV is set to development in your environment
  2. Use the static HTML documentation links above
  3. Check the console for any JavaScript errors
  4. Refer to the running and deployment guide for additional troubleshooting steps
